# aws에 s3를 활용한 외부 디스크 사용

## 사용이유: 
- 운영 난이도 + 가격 + 안정성 모든 면에서 s3가 좋다.

- s3는 용량을 계속 늘릴수 있다 는 점에서 무제한에 가깝다고 할 수 있다. 

- 99.999999999%에 내구성을 가지고 있다.

## 총 3개의 s3 bucket을 소유할 계획이다.

### clobby-local-disk

- local에서 테스트 할때 사용할 disk 이다. 

### clobby-test-server-disk

- test server에서 사용할 disk 이다.

### clobby-deploy-server-disk

- deploy server에서 사용할 disk 이다.
